# Todo Application

## Overview

This is a Todo Application that you can record your tasks.

![Initial look of the application](./assets/01.png)

You can pick a task from sidebard to display it on the tab.

![Application is displaying the selected task from sidebar on the tab](./assets/02.png)

You can add a new task

![Adding a new task modal is open](./assets/03.png)

![The added new task is displayed on tab](./assets/04.png)

You can remove the task if you want

![Delete task modal is open](./assets/05.png)

![When the task is deleted, it is removed both from tab and sidebar](./assets/06.png)
